# Pizzabot

### About
A conversational assistant to order pizza.

This chatbot was built using [Rasa](https://rasa.com/docs/getting-started/). All requirements are included in requirements.txt file.

### Train the chatbot
```
rasa train
```

### Test chatbot
```
rasa test
```

### Talk to chatbot
```
rasa shell
```
And on another terminal:
```
rasa run actions
```
